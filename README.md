# Project_S2
🅿️ Parking System (Entrée / Sortie)

Une application desktop moderne développée en **Python 3.11** pour la gestion simplifiée d'un parking. Ce système permet d'enregistrer les arrivées, les départs et de calculer automatiquement le temps de stationnement.

---

Fonctionnalités


* **Enregistrement des Entrées** : Capture l'heure exacte d'arrivée par matricule.
* **Gestion des Sorties** : Calcule automatiquement la durée de séjour dès la sortie du véhicule.
* **Log en Temps Réel** : Une console intégrée affiche l'historique des actions durant la session.
* **Calcul de Durée** : Gère correctement les passages de minuit (calcul sur 24h).
* **Persistance des données** : Sauvegarde automatique de chaque mouvement dans un fichier texte externe.

---

Les bibliotheques utilisées 


* **Tkinter** : Pour l'interface graphique (GUI).
* **Datetime** : Pour la gestion précise de l'horodatage.
